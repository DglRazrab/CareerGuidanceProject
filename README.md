# CareerGuidanceProject

Описание на русском:
Игра (кликер), включающая в себя мини-игры по некоторым профессиям. Создана совместно с художником по его задумке. Моя роль - программист и разработчик интерфейса. Ниже представлено несколько скриншотов.

Description in English:
A game (clicker) that includes mini-games of some professions. Created jointly with the artist according to his idea. My role is a programmer and interface developer. Below are a few screenshots.
